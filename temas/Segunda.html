Tipos de Variables y Constantes en PHP


1. Variables

Tipo Características Uso Nombres válidos Ejemplo
String Una cadena de caracteres. No tiene límite de longitud en teoría. Se usa para almacenar texto. Debe empezar con un símbolo de dólar ($), seguido de una letra o guion bajo. $nombre = "Juan";
Carga: echo $nombre;
Integer Números enteros, positivos o negativos. Se usa para almacenar números sin decimales. Debe seguir las reglas generales de las variables en PHP. $edad = 25;
Carga: echo $edad;
Float Números con punto decimal (flotantes). Se usa para almacenar números decimales. Debe seguir las reglas generales de las variables en PHP. $precio = 19.99;
Carga: echo $precio;
Boolean Puede tener dos valores: true o false. Se usa para lógica binaria. Debe seguir las reglas generales de las variables en PHP. $activo = true;
Carga: echo $activo;
Array Conjunto de valores almacenados en una única variable. Se usa para almacenar múltiples valores. Debe seguir las reglas generales de las variables en PHP. $frutas = array("manzana", "naranja", "mango");
Carga: echo $frutas[1];
Object Una instancia de una clase. Puede contener propiedades y métodos. Se usa para programación orientada a objetos. Debe seguir las reglas generales de las variables en PHP. class Persona { public $nombre = "Juan"; }
$persona = new Persona();
Carga: echo $persona-<nombre;
NULL Representa la ausencia de valor. Se usa cuando una variable no tiene valor. Debe seguir las reglas generales de las variables en PHP. $variable = NULL;
Carga: echo $variable;

2. Constantes

Nombre Características Uso Ejemplo
Constantes en PHP Las constantes no pueden cambiar su valor una vez asignadas. Se usan para valores fijos que no deben cambiar. define("PI", 3.1416);
Carga: echo PI;
Constantes mágicas Son predefinidas por PHP y cambian según el contexto. Se usan para obtener información del entorno de ejecución. echo __FILE__; (Muestra el archivo actual)
echo __LINE__; (Muestra la línea actual)
PHP_VERSION Contiene la versión de PHP en ejecución. Se usa para verificar la versión de PHP. echo PHP_VERSION;
PHP_OS Contiene el nombre del sistema operativo en el que PHP se está ejecutando. Se usa para obtener información del sistema operativo. echo PHP_OS;
DIRECTORY_SEPARATOR Contiene el separador de directorios para el sistema operativo actual. Se usa para construir rutas de archivos de forma segura. echo DIRECTORY_SEPARATOR;
PHP_INT_MAX Contiene el valor máximo de un entero en PHP. Se usa para verificar los límites de los enteros. echo PHP_INT_MAX;
PHP_EOL Contiene el carácter de salto de línea adecuado para el sistema operativo actual. Se usa para generar saltos de línea en una salida. echo PHP_EOL;

4. Operadores en PHP

Tipo Características Uso Ejemplo
Operadores Aritméticos Realizan operaciones matemáticas básicas. Se usan para realizar cálculos. $suma = 5 + 3;
$resta = 5 - 3;
$multiplicacion = 5 * 3;
$division = 5 / 3;
Operadores de Comparación Comparan dos valores y retornan un valor booleano. Se usan para comparar valores en condiciones. $igual = (5 == 3);
$mayor = (5 > 3);
$menor_o_igual = (5 <= 3);
Operadores Lógicos Operan sobre valores booleanos. Se usan para combinar múltiples condiciones. $y = (true && false);
$o = (true || false);
$no = !true;
Operadores de Asignación Asignan valores a las variables. Se usan para asignar o modificar valores de variables. $x = 5;
$x += 3; (equivalente a $x = $x + 3;)
Operadores de Cadena Operan sobre cadenas de texto. Se usan para concatenar cadenas. $texto = "Hola " . "Mundo!";
