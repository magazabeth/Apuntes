CONDICIONES EN PHP

regresar

Guía de condiciones en PHP En PHP, las condiciones permiten tomar decisiones dentro de un script según los valores de variables o el resultado de expresiones. Los bloques de código se ejecutan o no dependiendo de si una condición es verdadera (true) o falsa (false). Estructuras condicionales básicas en PHP:
  • 1. if
  • 2. if-else
  • 3. if-elseif-else
  • 4. switch
  • 5. Operador ternario
  • 6. Operador null coalescente

1. Estructura if
La estructura if ejecuta un bloque de código solo si una condición es verdadera. Sintaxis: 
if (condición) {
    // Código a ejecutar si la condición es verdadera
}
Ejemplo:
<?php
$edad = 20;
if ($edad >= 18) {
    echo "Eres mayor de edad.";
}
?>
2. Estructura if-else
La estructura if-else se usa cuando queremos ejecutar un bloque de código si la condición es verdadera, y otro bloque si la condición es falsa. 
if (condición) {
    // Código si la condición es verdadera
} else {
    // Código si la condición es falsa
}
Ejemplo:
<?php
$hora = 14;
if ($hora < 12) {
    echo "Buenos días.";
} else {
    echo "Buenas tardes.";
}
?>
3. Estructura if-elseif-else
La estructura if-elseif-else se utiliza cuando hay múltiples condiciones que deben ser evaluadas en orden. 
if (condición1) {
    // Código si la condición1 es verdadera
} elseif (condición2) {
    // Código si la condición2 es verdadera
} else {
    // Código si ninguna de las condiciones anteriores es verdadera
}
Ejemplo:
<?php
$calificacion = 85;
if ($calificacion >= 90) {
    echo "Excelente";
} elseif ($calificacion >= 80) {
    echo "Muy bien";
} elseif ($calificacion >= 70) {
    echo "Bien";
} else {
    echo "Necesitas mejorar";
}
?>
4. Estructura switch
La estructura switch se utiliza cuando una variable o expresión puede tener varios valores posibles y se desea ejecutar un bloque de código específico para cada valor. 
switch (expresión) {
    case valor1:
        // Código si la expresión es igual a valor1
        break;
    case valor2:
        // Código si la expresión es igual a valor2
        break;
    default:
        // Código si la expresión no coincide con ningún valor anterior
}
Ejemplo:
<?php
$dia = "lunes";
switch ($dia) {
    case "lunes":
        echo "Hoy es lunes.";
        break;
    case "martes":
        echo "Hoy es martes.";
        break;
    case "miércoles":
        echo "Hoy es miércoles.";
        break;
    default:
        echo "No es ni lunes, ni martes, ni miércoles.";
}
?>
5. Operador ternario
El operador ternario es una forma abreviada de escribir una condición if-else. 
condición ? valor_si_verdadero : valor_si_falso;
Ejemplo:
<?php
$edad = 20;
echo ($edad >= 18) ? "Eres mayor de edad." : "Eres menor de edad.";
?>
6. Operador Null Coalescente
El operador null coalescente (??) permite verificar si una variable está definida y no es null. Si la variable no está definida o es null, se evalúa el valor a la derecha del operador. 
$variable = expresión ?? valor_si_null;
Ejemplo:
<?php
$nombre = $_GET['nombre'] ?? 'Invitado';
echo "Hola, $nombre";
?>
Condiciones Complejas 
Las condiciones en PHP también pueden combinar múltiples comparaciones y operadores
lógicos (&&, ||, etc.) para evaluar varias expresiones a la vez.
Ejemplo con operadores lógicos:
<?php
$edad = 25;
$licencia = true;
if ($edad >= 18 && $licencia) {
    echo "Puedes conducir.";
} else {
    echo "No puedes conducir.";
}
?>

Conclusión El uso de condiciones en PHP es fundamental para construir lógica en tus aplicaciones. Existen varias formas de hacer decisiones basadas en comparaciones y operadores lógicos, adaptándose a diferentes casos. Las estructuras condicionales permiten controlar el flujo del programa de manera eficiente, asegurando que las acciones correctas se ejecuten bajo las circunstancias adecuadas.
