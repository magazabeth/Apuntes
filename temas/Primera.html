1. ¿Quien inventó PHP? En 1994, Rasmus Lerdorf, un ingeniero danés-canadiense, ideó PHP con un objetivo bastante modesto: quería monitorizar el tráfico de su currículum en línea. Empezó con un conjunto de scripts CGI escritos en el lenguaje de programación Perl. 2. ¿Qué significa PHP? PHP es un lenguaje de programación de código abierto que se utiliza para desarrollar aplicaciones y páginas web. PHP es un acrónimo recursivo para PHP : preprocesador de hipertexto. Es especialmente adecuado para el desarrollo web y puede ser incrustado en HTML. PHP favorece la conexión entre los servidores y la interfaz de usuario. Las diferencias más notables entre JavaScript y PHP son: PHP es un lenguaje de scripting del lado del servidor, mientras que JavaScript es un lenguaje de scripting del lado del cliente. PHP se desarrolla solamente en el backend de una página web, mientras que JavaScript fue creado como una solución para el frontend, aunque desde 2009 también dispone de entorno de tiempo de ejecución de backend. PHP se utiliza para ejecutar funciones del servidor, mientras que JavaScript se ejecuta dentro del navegador. PHP admite bases de datos, mientras que JavaScript no admite bases de datos. 4. ¿Qué tipo de licencia tiene PHP La licencia PHP es la licencia bajo la que se publica el lenguaje de programación PHP. Una vez que hayas creado tu archivo "index.html" y guardado tus archivos en la carpeta "htdocs", puedes acceder a tu sitio web o aplicación a través de la dirección "localhost" en tu navegador. Por ejemplo, si tienes un archivo llamado "index.html" en la carpeta "htdocs" y has iniciado Apache correctamente, puedes acceder a tu sitio web o aplicación ingresando "http://localhost" en la barra de direcciones de tu navegador. 8. ¿Dónde se encuentra la página de inicio de XAMPP? 10. Cómo instalar php en una memoria usb
  • Descargar PHP: Visita el sitio web oficial de PHP en https://www.php.net/downloads y descarga la versión de PHP que desees. Asegúrate de elegir la versión adecuada para tu sistema operativo (Windows, macOS, Linux) y que sea una versión portátil si es posible.
  • Extraer los Archivos: Conecta tu memoria USB a tu computadora. Extrae el archivo comprimido que descargaste en el paso anterior. Puedes usar una herramienta de extracción de archivos como WinRAR, 7-Zip, o cualquier otra herramienta compatible.
  • Copiar Archivos a la Memoria USB: Copia la carpeta extraída con los archivos de PHP a la raíz de tu memoria USB. Asegúrate de que todos los archivos y subcarpetas estén correctamente copiados.
  • Configurar PHP: Dentro de la carpeta de PHP en la memoria USB, busca el archivo `php.ini-development` o `php.ini-production`. Renómbralo a `php.ini` y edítalo según tus necesidades. Por ejemplo, puedes ajustar la configuración de `error_log`, `upload_max_filesize`, y otras opciones relevantes.
  • Probar PHP: Para probar que PHP funciona correctamente, abre una terminal o línea de comandos y navega hasta la carpeta de PHP en la memoria USB. Ejecuta el comando `php -v` para verificar que PHP está instalado y funcionando. También puedes crear un archivo de prueba llamado `info.php` con el siguiente contenido y ejecutarlo a través de un servidor web: 
    <?php
phpinfo();
?>
    Guarda el archivo `info.php` en la carpeta `htdocs` (si existe) o en una carpeta específica para tu servidor web y accede a él desde tu navegador para verificar que PHP está funcionando correctamente.
  • Uso de PHP en la Memoria USB: Puedes ahora utilizar PHP directamente desde la memoria USB en cualquier máquina compatible. Solo necesitas asegurarte de que el servidor web de la máquina tenga acceso a la carpeta de PHP en la memoria USB.
